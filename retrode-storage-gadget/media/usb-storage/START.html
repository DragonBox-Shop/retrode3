Welcome to Openpandora Retrode3

Getting started

  • connect Retrode 3 to some USB port (you must already have done if you have opened this file)
  • open the Web interface built into the Retrode3: http://192.168.0.202
  • follow the instructions you will find there.

Troubleshooting

  • If you see this page but no logo, enable Internet access in your Host-PC and Browser.
  • If you can't access http://192.168.0.202, make sure that your computer has RNDIS or ACM or NCM protocols (Internet over USB) enabled.
  • This should be automatic on Windows and macOS. On Linux you may have to install/load some kernel modules (rndis_host) and/or configure in a Network manager.
